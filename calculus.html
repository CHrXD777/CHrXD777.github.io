\(\sin{x}\) and \(\cos{x}\)

勾股定理：a2 + b2 = c2 → \(\frac{\frac{a^2}{c^2}}{\frac{b^2}{c^2}}\) = 1 → (\(\cos{\theta}\))2 +(\(\sin{\theta}\))2 = 1

证明

\(\displaystyle\lim_{\theta\to 0} \frac{\sin{\theta}}{\theta}\) = 1

1.\(\sin{\theta}\) < θ | θ > 0

2.\(\tan{\theta}\) > θ | 0 < θ < \(\frac{\pi}{2}\)

3.\(\frac{\sin{\theta}}{\cos{\theta}}\) > θ | \(\frac{\sin{\theta}}{\theta}\) > \(\cos{\theta}\)

\(\frac{\Delta\sin{x}}{\Delta x}\) = \(\frac{\Delta\sin{(x+\Delta x)} - \Delta\sin{\Delta x}}{\Delta x}\)

两角和公式：

\(\sin{(x+\Delta x)}\) = \(sin{x}\cos{\Delta x} + \cos{x}\sin{\Delta x}\)

等式两边同时变化：

\(\frac{\Delta\sin{(x+\Delta x)} - \Delta\sin{\Delta x}}{\Delta x} = \frac{\sin{x}\overbrace{(\cos{\Delta x - 1})}^{=0}}{\underbrace{\Delta x}_{①}} + \frac{\cos{x}\overbrace{\sin{\Delta x}}^{=1}}{\underbrace{\Delta x}_{②}}\)

对于②：已证为1；易知①为0，所以证得\(\sin{x}\)的导数为\(\cos{x}\)

逆函数、对数函数

指数函数的逆函数：

自然对数：\(x = \ln y\)

逆函数：\(y = f(x) \ and \ x = f^{-1}(y)\) -—只有双射才能有逆函数（必须单调？）且斜率是原函数斜率的倒数

性质：

  • \(\ln(y\cdot Y) = \ln y + \ln Y\)
  • \(\ln (y^n) = n\ln y\)

\(\ln y\) 和 \(\sin^-1(y) (arcsin)\)的导数

  • \(\frac{d}{dy} \ln y = \frac{1}{y}\)
  • \(\arcsin y = \frac{1}{\sqrt{1 - y^2}}\)

  • \(\arccos y = -\frac{1}{\sqrt{1 - y^2}}\)

    \(\frac{d}{dy}(\arcsin y + \arccos y) = 0\) \(\rightarrow\) \(\arcsin y + arccos y = \) 常数（\(\frac{\pi}{2}\)）

    函数与反函数就像一条路径，先A\(\rightarrow\)B,再B\(\rightarrow\)A

    由以上思想可得\(e^{\ln y} = y\)，两边求导验证对数的导数为\(\frac{1}{y}\)

    所以对\(\arcsin\)函数：\(x = \sin^{-1}y = \arcsin y\)，则\(y = \sin(sin^{-1}y)\)

牛顿法：\(x-a \approx \frac{f(a)}{f'(a)}\)

牛顿法可多次近似（a取前一次近似的结果）
